One-line: A scannable badge for a transaction outflow's kind — opposition (red), support (green), rescind (gold), transfer (stone).

```jsx
<OutflowBadge kind="support" />
<OutflowBadge kind="opposition" />
<OutflowBadge kind="transfer" />
```

Use in transaction/outflow tables. Colors follow the ledger's reserved semantics; rescind is gold (a correction), transfer is neutral stone.
